# Next JS w/ Styled Components Boilerplate

## Usage:

Install

To clone the repository:

```
    git clone ...
```

To install the project run:

```
    npm install

    or

    yarn
```

## Deployment

Install Now CLI: [Zeit Now CLI](https://zeit.co/download)

```
    npm i -g now
```

Using the CLI, in your root directory run:

```
    now
```
